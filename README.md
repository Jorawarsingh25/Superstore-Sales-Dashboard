# Superstore Sales Dashboard

## Project Overview
This project analyzes retail sales performance using the Superstore Sales dataset.

## Tools Used
- Power BI
- Excel
- Data Cleaning

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Sales Trend Analysis
- Category Analysis
- Regional Performance
- Customer Insights

- Technology category generated the highest sales.
- West region showed strong profitability.
